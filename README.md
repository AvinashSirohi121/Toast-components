# Toast Components 🚀

A lightweight and reusable Toast Notification Component built using **React** and **JavaScript**.

This project allows you to display different types of toast notifications such as:

- ✅ Success
- ℹ️ Info
- ⚠️ Alert
- ❌ Failure

The notifications appear in a stacked layout and can be dismissed manually.

## 📸 Preview

Toast notifications with different variants:

- Info Toast
- Alert Toast

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/AvinashSirohi121/Toast-components.git
```

### 2. Navigate into the project

```bash
cd Toast-components
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

---

## 🎯 Usage

Example:

```jsx
showToast({
  type: "success",
  message: "Task completed successfully!",
});
```

Supported types:

```js
success
info
alert
failure
```
